# Spotify ETL Pipeline (Go)

Go-based ETL pipeline for Spotify podcasts.

## Tech stack (Phase 0)

- Go 1.22
- API framework: chi
- Postgres: pgx
- Redis: go-redis/v9
- Kafka: kafka-go
- Observability: prometheus client, zap logging, OpenTelemetry
- OpenAPI tooling: oapi-codegen
